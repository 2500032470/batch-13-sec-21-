# batch-13-sec-21-
Student Performance Management System
Description

This is a Python program to manage and analyze student performance. It takes student names and marks as input, calculates total, average, and grade, and provides overall class analysis.

Features
Add multiple students
Calculate total and average marks
Assign grades (A, B, C, D, F)
Display student details
Find topper
Calculate class average
Count passed and failed students
Grade Criteria
90 and above → A
75 to 89 → B
60 to 74 → C
50 to 59 → D
Below 50 → F
How to Run
Install Python
Save the code in a file (example: main.py)
Run the program using:
python main.py
Example Input

Enter number of students: 2
Enter name of student 1: John
Enter marks (space separated): 85 90 80
Enter name of student 2: Alice
Enter marks (space separated): 70 75 72
